## ✨ Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Double-click text to edit
- Local Storage persistence
- Responsive design
- Total and completed task counters

## 📦 Installation

```bash
git clone https://github.com/nagesh1-12/react-todo-app.git
cd react-todo-app
npm install
npm run dev
```
